Задание
Напишите функцию PrintDocuments, принимающую вектор документов и два числа: 
skip_start и skip_finish. Пусть функция сортирует документы по убыванию рейтинга 
и выводит документы в таком порядке, пропустив skip_start документов с наибольшим
рейтингом и skip_finish с наименьшим. Считайте, что рейтинги не повторяются.
Числа на входе функции могут быть сколь угодно большими: ваша программа должна 
обрабатывать их корректно и не падать. Если требуется пропустить все документы 
или даже больше, не выводите ничего. Вектор документов может быть пустым, 
skip_start и skip_finish не обязательно меньше размера вектора. Формат вывода 
документов — в примере:
Пример

PrintDocuments(
    {
        {100, 5},
        {101, 7},
        {102, -4},
        {103, 9},
        {104, 1}
    },
    1,
    2
); 

Вывод

{ id = 101, rating = 7 }
{ id = 100, rating = 5 }
